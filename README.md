# Sematec University Power BI Dashboard

Power BI dashboard project for Sematec University analysis.

## Project File

- `Sematec University.pbix` - Power BI Desktop report file.
- `SEMATEC University.pdf` - exported report view for quick browsing.
- `1.png` through `6.png` - dashboard page previews.

The PBIX file is stored with Git LFS because it is larger than GitHub's standard single-file upload limit.

## How To Open

1. Install Git LFS if you are cloning the repository.
2. Open `Sematec University.pbix` with Microsoft Power BI Desktop.
